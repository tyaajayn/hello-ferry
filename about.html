What is HelloFerry?

HelloFerry is a prototype website developed for the Sabah-Labuan Online Ferry Ticket Booking System (SOFT) project by Nora Athira Yasmin Binti Kassim (S52121), in which its' main purpose is to provide a seamless online web-based ticket booking process for ferry/speed boat in Sabah and Labuan with a user friendly interface.

This site is a prototype that was done for final year project presentation and evaluation purposes.
